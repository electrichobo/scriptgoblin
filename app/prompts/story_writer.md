Format every element correctly:

- Action lines: present tense, flush left, describe only what can be seen or heard
- Character cues: character name only, all caps, on its own line before dialogue
- Parentheticals: (in parentheses), only when essential to reading
- Dialogue: the character's spoken words, below the character cue
- Transitions: FADE IN:, CUT TO:, FADE OUT. — use sparingly

Honor the project's theme, tone, and genre throughout.
Match length precisely to the target runtime — do not pad or cut short.
